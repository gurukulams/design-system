# மூன்றாம் தரப்பு ஒருங்கிணைப்புகள்
குருகுலம் எவ்வாறு வெளிப்புற சேவைகளுடன் தொடர்பு கொள்கிறது.

---

## இணைப்பு அடுக்கு (Connector Layer)
வெளிப்புற வினா வங்கிகளைச் செயலாக்க Node.js ஸ்கிரிப்ட்களைப் பயன்படுத்துதல்.

---

## GraphQL API
துல்லியமாக ஆவணங்கள் மற்றும் பயிற்சிகளை வினவுதல் (Querying).

---

## Oracle Cloud VM
நம்பகமான உள்கட்டமைப்பில் எங்களது துணை அமைப்புகளை நிலைநிறுத்துதல்.
