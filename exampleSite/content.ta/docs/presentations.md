குருகுலத்தின் **விளக்கக்காட்சித் தொகுதி (Presentation Module)** என்பது நிலையான ஆவணங்களை (static documentation) துடிப்பான, ஸ்லைடு-அடிப்படையிலான கற்றல் அனுபவமாக மாற்ற வடிவமைக்கப்பட்டுள்ளது. உள்ளடக்கத்தை உருவாக்குபவர்கள் எளிய Markdown கோப்புகளைக் கொண்டு வகுப்பறை கற்பித்தல், இணையவழி கருத்தரங்குகள் அல்லது காட்சி வழிக் கற்றலுக்குத் தேவையான ஸ்லைடுகளை உருவாக்க இது அனுமதிக்கிறது.

---

## விளக்கக்காட்சித் தொகுதி என்றால் என்ன?

ஒரு விளக்கக்காட்சி என்பது அடிப்படையில் ஒரு கிடைமட்டக் கோடு (horizontal rule) மூலம் பிரிக்கப்பட்ட Markdown பிரிவுகளின் தொகுப்பாகும். ஒவ்வொரு பிரிவும் ஒரு தனி ஸ்லைடாக மாறும். இந்தத் தொகுதியில் நாம் ஏற்கனவே பார்த்த எழுத்துரு, கணிதம் மற்றும் குறியீட்டுத் தொகுதி (code block) தரநிலைகள் அனைத்தும் செயல்படும். ஆனால் அவை முழுத் திரையில் தெளிவாகத் தெரியும் வகையில் மேம்படுத்தப்பட்டுள்ளன.

### முக்கிய அம்சங்கள்:
* **தானியங்கி ஸ்லைடு உருவாக்கம்:** PowerPoint போன்ற சிக்கலான மென்பொருட்கள் தேவையில்லை; எளிய Markdown குறியீடுகள் ஸ்லைடுகளை உருவாக்குகின்றன.
* **குறியீடு சார்ந்த தளவமைப்பு (Code-Centric Layouts):** இது மென்பொருள் உருவாக்குநர்களுக்கு ஏற்றது. பெரிய திரைகளிலும் குறியீடுகள் தெளிவாகத் தெரியும் வகையில் வண்ணமயமாகவும் பெரிய அளவிலும் காட்டப்படும்.
* **ஊடாடும் கூறுகள்:** பார்வையாளர்களின் ஈடுபாட்டைத் தக்கவைக்க, ஸ்லைடுகளுக்கு இடையில் வினாடி வினாக்கள் மற்றும் "சிறந்ததைத் தேர்ந்தெடு" வினாக்களை நேரடியாக இணைக்க முடியும்.

---

## ஸ்லைடுகளை உருவாக்கும் முறை

ஒரு ஸ்லைடை அடுத்த ஸ்லைடிலிருந்து பிரிக்க, புதிய வரியில் மூன்று கோடுகளை `---` பயன்படுத்த வேண்டும்.

### அடிப்படைத் தொடரியல் உதாரணம்:

```markdown
# ஜாவா அறிமுகம்
ஜாவா புரோகிராமிங் தொகுதிக்கு உங்களை வரவேற்கிறோம்.
- எளிமையானது (Simple)
- பாதுகாப்பானது (Secure)
- எங்கும் இயங்கக்கூடியது (Portable)

---

## ஒரு வகுப்பை (Class) வரையறுத்தல்
ஒவ்வொரு ஜாவா நிரலும் ஒரு கிளாஸ் வரையறையுடன் தொடங்குகிறது.

```java
public class Main {
    public static void main(String[] args) {
        System.out.println("Hello World");
    }
}
```

---

## கணித அடிப்படைகள்
சிக்கலான தர்க்கங்களையும் நாம் தெளிவாக முன்வைக்க முடியும்:

$$E = mc^2$$
```

---

## கற்றலில் விளக்கக்காட்சிகளின் முக்கியத்துவம்

1.  **கவனம் குவித்தல்:** தகவல்களைத் தனித்தனி ஸ்லைடுகளாகப் பிரிப்பதன் மூலம், ஒரே நேரத்தில் ஒரு கருத்தில் மட்டும் மாணவர்கள் கவனம் செலுத்த இது உதவுகிறது.
2.  **காட்சி வரிசைமுறை:** பெரிய தலைப்புகள் மற்றும் புல்லட் புள்ளிகள் ஒரு அறையின் கடைசியில் இருப்பவர் அல்லது சிறிய மொபைல் திரையில் பார்ப்பவர் கூட மையக்கருத்தைப் புரிந்துகொள்ள உதவுகின்றன.
3.  **படிபடியான விளக்கம்:** ஒரு மென்பொருளை நிறுவுவது போன்ற செயல்முறைகளை காலவரிசைப்படி விளக்குவதற்கு இது மிகவும் உகந்தது.

---

## உள்ளடக்கத்தை உருவாக்குபவர்களுக்கான சிறந்த நடைமுறைகள்

* **ஆறு வரிகள் விதி:** ஒரு ஸ்லைடில் ஆறு வரிகளுக்கு மேல் உரை இல்லாமல் பார்த்துக் கொள்ளுங்கள். தகவல் அதிகமாக இருந்தால், அதை இரண்டு ஸ்லைடுகளாகப் பிரிக்கவும்.
* **காட்சி இடைவெளிகள்:** அதிக உரை கொண்ட ஸ்லைடுகளுக்கு இடையில் வரைபடங்கள் அல்லது ஸ்கிரீன்ஷாட்களைச் சேர்க்க நாம் ஏற்கனவே பார்த்த **படங்கள் (Images)** தொடரியலைப் பயன்படுத்தவும்.
* **சீரான தலைப்புகள்:** ஒவ்வொரு ஸ்லைடிலும் `##` நிலை 2 தலைப்பு இருப்பதை உறுதி செய்யவும். இது தற்போது எந்த உட்பிரிவு விவாதிக்கப்படுகிறது என்பதை மாணவர்கள் அறிய உதவும்.
* **பயிற்சியுடன் முடித்தல்:** உங்கள் விளக்கக்காட்சியை எப்போதும் ஒரு "பயிற்சி" ஸ்லைடு அல்லது அடுத்த கட்ட நடவடிக்கைகள் குறித்த சுருக்கத்துடன் முடிக்கவும்.
