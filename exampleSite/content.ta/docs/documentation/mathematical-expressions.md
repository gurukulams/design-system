கல்வி சார்ந்த தளங்களுக்கு, கணிதக் குறியீடுகளைத் தெளிவாகவும் துல்லியமாகவும் வழங்குவது மிகவும் அவசியம். **குருகுலம் வடிவமைப்பு அமைப்பு (Gurukulams Design System)** கணித சூத்திரங்களைக் காண்பிப்பதற்கு ஒரு நிலையான முறையைப் பயன்படுத்துகிறது. இது சூத்திரங்கள் பார்ப்பதற்கு நேர்த்தியாகவும், உரையின் வரிசையோடு சரியாக அமைவதையும், எந்தத் திரையிலும் எளிதாக வாசிக்க முடிவதையும் உறுதி செய்கிறது.

---

## கணிதக் குறியீடுகளை எழுதுதல்

உங்கள் ஆவணங்களில் கணிதத்தைச் சேர்க்க உங்களுக்குத் தனி மென்பொருள் எதுவும் தேவையில்லை. ஒரு எளிய முறையைப் பயன்படுத்தி உங்கள் உரை கோப்புகளிலேயே நேரடியாக சூத்திரங்களை எழுதலாம்.

### 1. வரிக்குள் அமையும் குறியீடுகள் (Inline Expressions)
ஒரு வாக்கியத்தின் உள்ளேயே கணிதக் குறியீடு அல்லது சிறிய சூத்திரத்தைச் சேர்க்க விரும்பினால், அதை ஒற்றை டாலர் குறிகளுக்குள் `$ ... $` இட வேண்டும்.

**உதாரணம்:**
வட்டத்தின் பரப்பளவுக்கான சூத்திரம் $A = \pi r^2$ ஆகும், இதில் $r$ என்பது ஆரம் (radius).

---

### 2. தனித்துத் தெரியும் குறியீடுகள் (Display Expressions)
உங்களுக்கு ஒரு சிக்கலான சூத்திரம் இருந்து, அது முக்கியத்துவத்திற்காக ஒரு தனி வரியில் மையமாகத் தெரிய வேண்டும் என்றால், அதை இரட்டை டாலர் குறிகளுக்குள் `$$...$$` இட வேண்டும்.

**உதாரணம்:**
ஒரு பல்லுறுப்புக் கோவையின் மூலங்களைக் (roots) கண்டறிய இருபடிச் சூத்திரம் (quadratic formula) பயன்படுத்தப்படுகிறது:

$$x = \frac{-b \pm \sqrt{b^2 - 4ac}}{2a}$$

---

### 3. பொதுவான சின்னங்கள் மற்றும் வடிவம்
அறிவியல் மற்றும் கணிதத்தில் பயன்படுத்தப்படும் நிலையான குறியீடுகளை நமது அமைப்பு ஆதரிக்கிறது. சில உதாரணங்கள் கீழே:

| முடிவு (Result) | எழுதும் முறை |
| :--- | :--- |
| **அடுக்குகள்** ($x^2$) | `x^2` |
| **கீழொட்டுகள்** ($H_2O$) | `H_2O` |
| **பின்னங்கள்** ($\frac{1}{2}$) | `\frac{1}{2}` |
| **வர்க்கமூலம்** ($\sqrt{x}$) | `\sqrt{x}` |
| **கிரேக்க எழுத்துக்கள்** ($\alpha, \beta, \Delta$) | `\alpha, \beta, \Delta` |

---

## இந்த முறையை ஏன் பயன்படுத்த வேண்டும்?

* **சீரான தன்மை (Consistency):** தளம் முழுவதும் உள்ள அனைத்து சூத்திரங்களும் ஒரே மாதிரியான எழுத்துரு மற்றும் இடைவெளியைக் கொண்டிருக்கும்.
* **உயர்தரம் (High Quality):** சூத்திரங்கள் திசையன் கிராபிக்ஸ் (vector graphics) முறையில் உருவாக்கப்படுவதால், பயனர்கள் பெரிதாக்கிப் பார்த்தாலும் அவை மங்கலாகத் தெரியாது.
* **தேடல் வசதி (Searchable):** படங்களாக இருக்கும் கணிதத்தைப் போலன்றி, இவை உரை வடிவில் இருப்பதால் தேடுபொறிகளால் (search engines) எளிதாகக் கண்டறிய முடியும் மற்றும் பார்வையற்றோருக்கான மென்பொருட்களும் இவற்றை வாசிக்க முடியும்.

## சிறந்த நடைமுறைகள்
* **எளிமையாக வைக்கவும்:** சுருக்கமான குறிப்புகளுக்கு "Inline" முறையையும், மாணவர்கள் உற்று கவனிக்க வேண்டிய சூத்திரங்களுக்கு "Display" முறையையும் பயன்படுத்தவும்.
* **அதிகப்படியாகப் பயன்படுத்த வேண்டாம்:** உண்மையான கணிதக் குறியீடுகளை மட்டும் டாலர் குறிகளுக்குள் வைக்கவும். சாதாரண முக்கியத்துவத்திற்கு தடிமனான (bold) அல்லது சாய்வான (italic) எழுத்து முறையைப் பயன்படுத்தவும்.
* **அடைப்புக்குறிகளைச் சரிபார்க்கவும்:** பின்னங்கள் போன்ற சிக்கலான பகுதிகளில் எண்களைக் குழுவாக்க வளைந்த அடைப்புக்குறிகள் `{ }` பயன்படுத்தப்படுகின்றன. ஒரு திறக்கும் குறி இருந்தால், அதை மூடுவதற்கான குறியும் இருப்பதை உறுதி செய்யவும்!
