# எழுத்துருக்கள் (Typography)
தெளிவாக வாசிப்பதற்காக நாங்கள் சீரான எழுத்துருக்களைப் பயன்படுத்துகிறோம்.
## தலைப்புகள்
பக்கத் தலைப்புகளுக்கு H1-யும், துணைப் பிரிவுகளுக்கு H2-H4-யும் பயன்படுத்தவும்.
