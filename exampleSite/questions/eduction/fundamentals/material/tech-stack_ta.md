## குருகுலம் (Gurukulams) வடிவமைப்பு அமைப்பில், தொழில்நுட்பத்தை அதன் குறிப்பிட்ட பங்குடன் பொருத்துக.

```markdown
எங்கள் கட்டமைப்பு வடிவமைப்பில், ஒவ்வொரு கருவியும் ஒரு குறிப்பிட்ட காரணத்திற்காக தேர்வு செய்யப்பட்டுள்ளது:

* **Bootstrap 5:** தனிப்பயன் CSS இல்லாமல் அனைத்து HTML அமைப்பையும் மற்றும் UI வடிவமைப்பையும் கையாளுகிறது.  
* **Rollup.js:** எங்கள் `Practice Maker` நூலகத்தை UMD மற்றும் ESM வடிவங்களில் தொகுக்கிறது.  
* **KaTeX:** இணையத்திற்கான மிக வேகமான கணித வடிவமைப்பை (typesetting) வழங்குகிறது.  

*குறிப்பு: Tailwind CSS ஒரு தவறாக சேர்க்கப்பட்ட தேர்வு (distractor) ஆகும்; இது எங்கள் தற்போதைய முக்கிய தொழில்நுட்பங்களில் ஒன்று அல்ல.*
```
